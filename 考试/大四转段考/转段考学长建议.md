# 原则
尽早开始，多轮复习，刷题为重
300 题/180 min
中内+诊断：50%
中基中药中诊方剂针灸：40%
四大经典+卫生法（3~5）：10%


# 资料准备
大纲+中医职业指导用书+蓝基因/医考帮

# 策略
一、重点抓中内、诊断 

二、中基、中诊、中药、方剂常规复习 

三、针灸:
循行经穴、主穴常规复习；
特殊穴位(如五输穴、络穴、郗穴等)、
配穴(自己总结+口诀) 
`针灸配穴趣味记忆总结，手三阴内缺通，手三阳外立正`

四、经典(内经伤寒金匮温病)卫生法规轻松复习

# 方法
一、先大纲+中医执业医生指导用书，先看再刷题

二、蓝基因:先刷题，不会再翻书；重复刷、看错题试卷

三、整理错题，常看常回顾四、自我测试:章节真题乱序/年份真题 

邪修:刷题

# 技能考试
临床预备课+看视频（B 站）+实操

# 小建议
最少留有 1 个月的时间复习
实操问诊记得舌脉
